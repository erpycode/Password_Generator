# تولید کننده رمز عبور 🔐

ابزاری ساده و زیبا برای تولید رمز عبور قوی در پایتون، کپی خودکار به کلیپ‌بورد و ذخیره اختیاری در فایل متنی.

🌐 [رفتن به نسخه انگلیسی](README.md)

---

## عملکرد برنامه

- از کاربر طول رمز عبور بین **۸ تا ۲۰** کاراکتر را می‌پرسد.
- از ماژول امن `secrets` برای تولید تصادفی کاراکترها استفاده می‌کند.
- رمز عبور را از حروف کوچک، حروف بزرگ، اعداد و نمادها می‌سازد.
- رمز تولید شده را روی نمایشگر چاپ می‌کند.
- رمز را به صورت خودکار با `pyperclip` در کلیپ‌بورد قرار می‌دهد.
- در صورتی که کاربر بخواهد، رمز را در فایل `Password.txt` ذخیره می‌کند.

## نیازمندی‌ها

- پایتون ۳
- بسته `pyperclip`

برای نصب بسته:

```bash
pip install pyperclip
```

## نحوه اجرا

از پوشه پروژه دستور زیر را اجرا کنید:

```bash
python Password_genartor.py
```

سپس مراحل زیر را انجام دهید:

1. طول رمز عبور مورد نظر را وارد کنید (`8` تا `20`).
2. برنامه رمز عبور امن تولید و نمایش می‌دهد.
3. رمز به‌طور خودکار در کلیپ‌بورد کپی می‌شود.
4. اگر `y` را وارد کنید، رمز در `Password.txt` ذخیره می‌شود.

## نکات

- اگر عدد خارج از بازه قانونی وارد کنید، برنامه دوباره از شما می‌خواهد مقدار صحیح را وارد کنید.
- رمز تولید شده ممکن است شامل نمادهای ویژه باشد؛ در مکان‌هایی که همه نمادها پشتیبانی نمی‌شوند، دقت کنید.
- هر بار که ذخیره را انتخاب کنید، محتویات فایل `Password.txt` بازنویسی می‌شود.
